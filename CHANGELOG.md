# CHANGELOG

All notable changes to this project should be documented in this file.

## [v1.1.4] - 2026-04-11

- 拆分热力图 SVG 模板为 `heatmap_year_template.svg` 与 `heatmap_month_template.svg`，分别优化年视图与月视图布局。
- 个人热力图标题现在优先显示注册时填写的 `extra_id`，否则回退到姓名或学号。
- 更新渲染层逻辑，按 `chart_type` 自动选择对应热力图模板。

## [v1.1.3] - 2026-04-11

- 调整热力图 SVG 输出：默认使用更大画布，宽度随年/月视图自动适配，避免模糊。
- 个人热力图标题改为使用成员姓名或学号显示，并移除“热力图”字样。
- 精简图例布局，去掉“贡献度”标签，修正图例间距。

## [v1.1.2] - 2026-04-11

- 重构热力图逻辑，将参数解析与数据构建从 `main.py` 提取到 `heatmap.py`。
- 优化热力图 SVG 样式，修正边缘模糊和标签重叠问题。

## [v1.1.1] - 2026-04-11

- 更新插件元数据，补充主页、文档链接、许可证和标签信息。
- 支持 `/rkc` 全能榜结果以图片形式发送。
- 新增 `/map` / `/热力图` 指令，支持个人与社团当前月与全年度 GitHub 风格活跃热力图。
