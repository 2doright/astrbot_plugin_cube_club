本插件是专为 **魔方社团/协会 (Magic Cube Association)** 设计的 AstrBot 插件。它通过打通聊天机器人的便捷交互，为社团成员提供一个集成绩记录、PB 统计、内部赛事管理于一体的轻量化系统。

## 🌟 核心特性

- **多域隔离机制 (Scoped Data)**：
  - **日常练习域 (Daily)**：用于社团成员日常练习打卡，支持灵活的日/月/年排行榜。
  - **赛事管理域 (Competition)**：管理员可创建独立的赛事域（如 `/comp25`），确保正式比赛成绩与练习记录互不干扰。
- **专业成绩统计**：
  - **PB 自动追踪**：自动记录并提示个人历史最优成绩。
  - **滚动均值逻辑**：依据项目特征自动识别并计算 **Ao5 (Average of 5)** 或 **Mo3 (Mean of 3)** 的当前及最优值。
  - **全量均值 (AoAll)**：追踪用户在特定域下的整体练习水平。
- **极致的交互体验**：
  - **双语指令**：支持中英文别名（如 `/rkd` = `/日榜`），照顾不同用户习惯。
  - **智能识别**：支持 `MM:SS.ss` 及纯秒数输入。
- **高性能底层**：基于 SQLite 预编译 SQL 及索引优化，保障数万级并发录入与毫秒级排行检索。

## 🎮 深度指令指南

### 1. 成员生态
| 指令 | 别名 | 格式 | 说明 |
| :--- | :--- | :--- | :--- |
| `/reg` | `注册` | `/reg <姓名> [ID]` | 绑定 QQ 与社团姓名，ID 可用于标记学号或 WCAID |
| `/me` | `我` | `/me` | 查看自己当前绑定的社团身份 |
| `/who` | `查人` | `/who <姓名/QQ>` | 根据姓名或 QQ 号反查成员信息 |

### 2. 成绩录入流水线
*   **日常打卡**：
    直接在群内输入：`/<项目代码> <成绩内容>`
    *   示例：`/333 12.55`
    *   示例：`/sq1 1:23.45`
*   **内部赛录入 (管理员专用)**：
    格式：`/comp[年份] <项目> <姓名> <成绩>`
    *   示例：`/比赛 333 张三 9.80` (记入当年赛事域)
    *   示例：`/comp24 444 李四 42.11` (补录 24 年历史成绩)

### 3. 多视角排行榜 (Ranking)
格式：`/rk[分类] <项目> <统计指标>`
*   **分类标识**：
    *   `d` (日): `/rkd`, `/日榜`
    *   `m` (月): `/rkm`, `/月榜`
    *   `y` (年): `/rky`, `/年榜`
    *   `a` (总): `/rka`, `/总榜`
    *   `c` (量): `/rkc`, `/勤奋榜` (按记录条数排序)
*   **指标参数**：`pb` (单次最高), `ao5` / `mo3` (最优平均), `aoAll` (历史均值)

## 🧩 支持项目概览

覆盖全部 WCA 官方项目及高频非官方项目：
*   **速拧系列**：`222` 至 `777`
*   **异形系列**：`sq1`, `clock` (钟), `minx` (五魔), `pyram3` (金字塔), `skewb` (斜转)
*   **盲拧/其他**：`222bf`, `333bf`, `333oh`, `hrd` (华容道)

## 🛠️ 抢先体验 (手动安装)

由于功能尚未完结，目前仅支持手动部署：

1. 进入 AstrBot 根目录下的 `data/plugins/` 文件夹。
2. 执行 Git 克隆指令：
   ```bash
   git clone https://github.com/2doright/astrbot_plugin_cube_club.git
   ```
3. 重启 AstrBot 即可。
